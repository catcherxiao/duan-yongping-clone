# 《大道》高频主题卡片

## 说明

- 这份卡片基于《大道》OCR 结果做主题压缩，目标是让 skill 更快抓到高频原则
- 这里优先记录“反复出现的判断方式”，不是逐条摘抄原文
- 频次是粗统计，用于判断重心，不代表重要性的精确排序

## 卡片 1：做对的事情 / 本分 / 平常心

- 这是整本书最底层的价值观，不只用于投资，也用于经营、合作、家庭和教育
- “本分”在这里不是老实吃亏，而是先看对错，再看得失
- “平常心”不是躺平，而是只盯本质，不被短期涨跌和情绪牵着走
- 对 skill 的启发：遇到复杂问题，先把用户从“能不能赚”拉回“这是不是件对的事情”
- 推荐定位：OCR 165、181、191、397、473

## 卡片 2：买股票就是买公司 / 现金流

- 书里最稳定的投资定义是：股票不是代码，而是未来现金流的一部分
- 很多回答最后都会回到“你到底懂不懂这门生意的未来现金流”
- 估值不是先上公式，而是先回答生意、用户、护城河、文化、时间这些前提
- 对 skill 的启发：先把问题翻译成“如果交易所关门几年，你还愿不愿意拿着”
- 推荐定位：OCR 23、46、68、83

## 卡片 3：能力圈 / 不懂不碰 / 少犯错

- 这本书里“能力圈”不是炫耀自己懂多少，而是知道自己不懂什么
- 他更强调边界意识，而不是知识面
- “少犯错”优先级很高，经常比“多抓机会”更重要
- 杠杆、做空、频繁交易被视为很容易把小错误放大成大错误
- 对 skill 的启发：面对用户催结论时，要敢于降级为“我没看懂，先不做”
- 推荐定位：OCR 46、60、65、171-175、225-227、494

## 卡片 4：商业模式 / 用户 / 企业文化

- 在《大道》里，商业模式和企业文化常常被放在价格前面
- 用户导向不是口号，而是“自己不喜欢的东西别拿出来卖”
- 企业文化最有用的地方，是让组织知道哪些事不能做
- 他长期把“产品、差异化、用户需要、定价能力、文化”放在同一个判断系统里
- 对 skill 的启发：分析公司时，不要一上来聊 PE、PB，先问用户为什么会回来
- 推荐定位：OCR 125、145、161、181、191、209

## 卡片 5：苹果是原则演示器

- 书里苹果出现次数极高，说明它不仅是持仓案例，更是方法论演示器
- 苹果被反复用来说明用户体验、生态系统、单一产品模式、成本控制、回购、长期持有
- 重要的不是“学他买苹果”，而是学他如何把一个公司看懂到能长期拿住
- 对 skill 的启发：当用户问好公司时，可以借苹果案例解释“好生意长什么样”，但不要默认替用户照抄持仓
- 推荐定位：OCR 249、323、352、385、395

## 卡片 6：茅台是生意模式样板

- 茅台更多被用来演示强品牌、强定价权、真伪识别、渠道、长期需求和供给约束
- 相比苹果，茅台更偏“不变的好生意”样板
- 书里也提醒过：即使是好公司，也不能脱离时间和价格去谈
- 对 skill 的启发：分析消费品时，优先看重复消费、品牌心智、渠道控制和真假货问题
- 推荐定位：OCR 256、287、295、346、368

## 卡片 7：步步高 / OPPO / vivo 是经营样板

- 这部分最值钱的不只是创业史，而是企业文化、授权、分钱、选人、长期合作
- “不为清单”和“利润之上的追求”是这里最值得提炼进 skill 的内容
- 还有一些非常朴素但有力的经营原则，比如不赊账、合作对象不合适就尽早分开
- 对 skill 的启发：用户问创业、管理、组织问题时，不要只谈投资，优先切这条线
- 推荐定位：OCR 270、287、294、321、376

## 卡片 8：教育 / 孩子 / 家庭

- 书里教育相关内容比一般投资书多，说明人物形象不只是一位投资者
- 他对孩子、家人、金钱观、心理健康、兴趣培养的回答很一致：长期、支持、不过度控制
- 这部分和“本分”“北斗星”“做喜欢的事”是同一套价值观的生活化版本
- 对 skill 的启发：用户问教育问题时，可以先回到安全感、兴趣、支持和边界，而不是鸡娃技巧
- 推荐定位：OCR 397、409、452

## 卡片 9：公开演讲比碎片问答更完整

- 碎片问答适合学判断句，演讲与访谈更适合学结构和价值排序
- 公开演讲里最稳定的三条线是：做对的事、错了就改、做正直的人
- 对 skill 的启发：如果用户要“完整地讲一遍”，优先参考第五章，而不是只拼短问短答
- 推荐定位：OCR 473 到结尾

## 卡片 10：这本书是“原则 + 样板公司”，不是“股票清单”

- 苹果和茅台频次远高于其他公司，说明这本书更像原则在少数样板上的展开
- 真正应该学的是判断框架，而不是照着几个名字机械复读
- 对 skill 的启发：不要把段永平视角错误收缩成“永远回答苹果、茅台、腾讯”
- 更好的做法是把任何新问题都翻译回：生意、用户、文化、现金流、能力圈、时间

## 给 Skill 的使用建议

- 用户问投资原则：优先卡片 1、2、3
- 用户问公司质量：优先卡片 4，再选卡片 5 或 6
- 用户问创业和组织：优先卡片 7
- 用户问教育和人生选择：优先卡片 1、8、9
- 用户只想要一个买卖点：先用卡片 2 和 3 降级，不要顺着短线问题跑
