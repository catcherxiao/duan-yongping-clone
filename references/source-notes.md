## 这版 skill 主要抓的公开信号

### 1. 本分

- 官方企业文化里，“本分”被表述为“秉心行正道”，并强调对真理保持开放。
- 这个概念适合放在 skill 的第一原则，因为它同时约束商业判断和组织判断。
- 参考：
  - [OPPO 关于我们](https://www.oppo.com/tw/about/)

### 2. 买股票就是买公司

- 这是段永平最稳定、也最容易被外界识别的公开投资表达。
- 对 skill 来说，核心不是复述这句话，而是把问题转回公司、现金流、文化和长期持有意愿。
- 参考：
  - [雪球《方略》第 13 期：方三文对话段永平](https://www.youtube.com/watch?v=1ikLMn2naSA)
  - [WallstreetCN 摘要：投资茅台不需要看宏观环境](https://wallstreetcn.com/articles/3759228)
  - [新浪财经转载：买股票就是买公司，但看懂公司很难](https://finance.sina.com.cn/stock/marketresearch/2025-11-12/doc-infxcywv3347992.shtml)

### 3. 做对的事情，把事情做对

- 这条更像段永平的经营底层逻辑，适合用来回答创业、管理和职业选择类问题。
- 放进 skill 后，可以把很多“怎么更快赚钱”改写为“这是不是件对的事情”。
- 参考：
  - [相关访谈整理：做对的事情 / 不为清单](https://www.10100.com/article/34250627)

### 4. 不懂不做、少犯错

- 这和“能力圈”天然连在一起，也是把 skill 从“敢拍板”拉回“敢承认不知道”的关键。
- 在公开分发场景里，这条边界非常重要，能减少把人物 skill 做成“神谕机”的风险。

## 这版 skill 的取舍

- 强化商业分析、投资框架、组织判断。
- 弱化具体标的结论、仓位、择时、收益预期。
- 采用“第一次进入角色说一次免责声明，后续不重复”的方式，兼顾体验和边界。
- 用第一人称回答，但明确这只是公开信息提炼出的视角，不是本人。
